About AI Assistant

The AI Assistant allows you to use natural language to:

  • Generate test data with specific parameters
  • Delete specific types of test data
  • Perform complex operations with simple commands
Example Commands:
  • "Create test data for 5 employees with night shifts in June 2025"
  • "Delete all test attendance logs"
  • "Generate attendance data with overtime scenarios for weekend work"
Your OpenAI API key is stored securely but will incur charges on your OpenAI account when the assistant is used.
